# RGR1_mathematics_analiz
# Численное интегрирование: методы Ньютона-Котеса

РГР №1 по численным методам.

## Что делает программа

Считает интегралы четырьмя методами:
- Прямоугольники
- Трапеции
- Симпсон
- Три восьмых

## Файлы

| Файл | Зачем |
|------|-------|
| `main.py` | Запустить это |
| `methods.py` | Формулы методов |
| `plots.py` | Графики погрешности |
| `convergence.py` | График сходимости |
| `README.md` | Этот файл |

## Как запустить

```bash
pip install numpy matplotlib
python main.py
```

## Что получится

5 картинок:
- `graph_smooth_*.png` — для функции eˣ
- `graph_rough_*.png` — для функции |x−0.5|
- `convergence_plot.png` — сходимость методов
